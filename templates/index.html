Description

This is the milestone project that I am building for the “Practical Python” stream, part of the New LMS in the “Full Stack Web Development Course” offered by Code Institute.

The purpose of the game is to guess a word or phrase from a given picture. Three attempts are allowed. With each subsequent attempt further clues will be shown at the expense of reduced points.

More information can be found in the About section of this game.

Instructions

First you need to register if you have not already done so.

If you have already registered, use your username to login. Your username is unique and its sole purpose is to keep track of the games you played and the points you gained. The user name can be any set of letters.
Once you log in you will be taken to your user page. There you can see a list of games played and the points gained. You can start a new game from this page.

More information can be found in the About section of this game.
